CSS Selectors and Specificity

CSS Selectors

Selectors Definition Example
Type Used to match all elements of a given type or tag name p, h1, span
Class This selector matches elements based on the contents of their class attribute. Use a prepend. .line, .box, .calendar
ID This selector matches elements based on the contents of their id attribute, which should be unique in the entire DOM. A # needs to be prepended. #book, #gold, #table

Selector Specificity

Ranking Selector Comments
1 ID Selector with the highest specificity
2 Class Lower specificity than ID but more specificit than type
3 Type Lowest specifity of the three
